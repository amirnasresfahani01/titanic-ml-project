# Notebooks

This folder contains the Jupyter notebooks used in the Titanic machine learning project.

Each notebook represents a specific step in the data science workflow:

- `01_data_understanding.ipynb` — initial dataset inspection
- `02_eda.ipynb` — exploratory data analysis
- `03_feature_engineering.ipynb` — feature creation and analysis
- `04_baseline_model.ipynb` — baseline Logistic Regression model
- `05_model_comparison.ipynb` — comparison of multiple machine learning models
